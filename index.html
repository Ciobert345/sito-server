Benvenuto a Manfredonia

Un'esperienza di gioco unica basata sul modpack BetterMinecraft Fabric Edition, che combina le migliori mod di avventura, tecnologia e magia. Esplora un mondo vasto e personalizzato, costruisci sistemi complessi e affronta sfide epiche in un ambiente amichevole e moderato.

BetterMinecraft offre un'esperienza di gioco completamente rinnovata con:

  • Nuove dimensioni e biomi da esplorare
  • Sistemi di magia avanzati con mod come Botania e Ars Nouveau
  • Tecnologia avanzata con Create e Applied Energistics
  • Miglioramenti grafici e di performance
  • Nuove creature e boss da combattere
  • Sistemi di progressione e crafting espansi

FAQ Tecniche

Caricamento Chunk

BetterMinecraft include 358 mod che aggiungono nuovi contenuti. Quando esplori una nuova area, il server deve generare tutti questi contenuti gradualmente.

Velocità Server

No! Una volta che un\'area è stata esplorata, il caricamento successivo sarà molto veloce. Il server è ottimizzato per performance fluide.

Generazione Mondo

Il nostro server utilizza tecniche avanzate di pre-generazione e ottimizzazione per bilanciare la qualità del mondo con le performance.

Migliorare Performance

Alloca 6-8GB di RAM, installa Optifine/Sodium, riduci la distanza di rendering a 12-16 chunk e disattiva animazioni non essenziali.

Crash e Errori

Verifica Java 17+, controlla l\'installazione delle mod, riavvia il launcher/gioco, e se necessario controlla i log o contatta un admin.

Mod Consigliate

Optifine/Sodium per le performance, JourneyMap per la navigazione, Inventory Tweaks per gestire gli inventari, e WAILA per identificare blocchi e oggetti.

Pannello di controllo

Il nostro server dispone di un pannello di controllo web che permette ai membri di monitorare lo stato del server e interagire con esso. Ecco come utilizzarlo:

Come accedere al pannello

  1. 1 Registrazione: Dopo l'approvazione della tua richiesta di whitelist, riceverai le credenziali di accesso al pannello.
  2. 2 Accesso tramite Discord: Entra nel nostro server Discord e vai al canale "link-bot-🤖".
  3. 3 Login al pannello: Utilizza il link fornito nel canale Discord per accedere al pannello e inserisci le tue credenziali.
  4. 4 Visualizzazione server: Una volta effettuato l'accesso, vedrai la lista dei server disponibili.

Gestione del server

  1. 1 Accedi alla dashboard: Dalla lista dei server, clicca su "View Dashboard" per accedere al pannello di controllo del server.
  2. 2 Controllo server: Nella dashboard, potrai visualizzare la console del server e monitorare le attività in tempo reale.
  3. 3 Gestione alimentazione: In alto a destra troverai un pulsante che ti permetterà di avviare, spegnere o riavviare il server.

⚠️ Importante

Utilizza le funzioni di avvio e spegnimento del server con parsimonia. Un uso improprio può causare danni ai dati di gioco e disturbare l'esperienza degli altri giocatori. Non spegnere mai il server senza prima avvisare gli altri utenti online.
